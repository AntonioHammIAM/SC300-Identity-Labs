# Lab 1 - Tenant Configuration

## Business Scenario

Contoso Manufacturing has migrated to Microsoft Entra ID.

As the Identity Administrator, your responsibility is to configure the tenant according to enterprise security standards.

---

## Objectives

- Configure company branding
- Review tenant properties
- Configure emergency access accounts
- Review licensing
- Validate security defaults

---

## Environment

- Microsoft Entra ID
- Microsoft 365 Developer Tenant

---

## Tasks

1. Sign into the Microsoft Entra Admin Center.
2. Review tenant information.
3. Configure company branding.
4. Verify domain configuration.
5. Review Security Defaults.
6. Create emergency access account.

---

## Validation

- Tenant information reviewed
- Company branding configured
- Emergency account documented

---

## Skills Demonstrated

- Microsoft Entra ID
- Tenant Administration
- Identity Governance
